ant design vue渲染表格显示no data

发表于 2019-12-02 | 分类于 ant design vue

按照ant design vue给出例子写表格组件，有表格数据，却渲染不出来，在分析这一问题，发现几个需要注意的点，做一记录

  • 一般开发环境，当出现问题时，都会给出警告信息，所以留意控制台的输出

  • 开发环境script引入vue注意引入方式如下，否则不能看到警告信息；需引入vue.js,而不是vue.min.js

  • 当表格作为组件时，表格数据属性不能用dataSource,而应该用data-source;否则不能正确渲染数据

数学对象常用处理数据的方法

发表于 2019-11-10 | 分类于 js

项目在需要对数据做处理的时候，经常需要百度一些最简单的数组、字符串方法，很是浪费时间；对这些方法经常回顾、归纳、总结，coding可以省下不少时间，本文章将会对常用处理数据场景及方法做总结，希望对你有所帮助。

方法归纳

  • abs() 取绝对值

  • round() 四舍五入

  • ceil() 向上取整

  • floor() 对数进行向下取整

  • max() 取最大值 max(nub1,nub2,nub3)

  • min() 取最小值

  • random() 取随机数

  • pow() Math.pow(x,y) 返回x的y次幂平方根

  • sqrt() 平方根 不可取负值，不可表根号

  • toFixed() 指定小数四舍五入后保留的位数

  • sin(x) 只写弧度x

  • PI
